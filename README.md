# Chat Server

## Compiled on Linux Ubuntu

### Server

```
    g++ server.cpp api/api.cpp -o server
```

### Client

```
    g++ -pthread client.cpp -o client
```

## Run

### Server

Before running:

```
    sudo apt install fortune-mod
```

Then:

```
    server run instructions here
```

### Client

```
    client run instructions here
```
